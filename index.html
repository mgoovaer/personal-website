REDEFINING NYC TWEETS Spring 2013
An optional class project for the undergraduate students in Large-Scale Data Analysis. Working with a graduate student, we developed an application that gathers and sorts tweets into select topics based on key words. The tweets were gathered using the Twitter API with a geocode delimiter limiting them to the NYC area. After manually categorizing a set of tweets into our ten predefined topics. We ran this training data set through Apriori to identify rules that would categorize the tweets based on words found in them. After some further tweaks we had a set of rules that could categorize the set of twitter data between 47% and 100% - dependent on the category. The remainder of the tweets were categorized. These results were then mapped, identifying geographical trends. Java
